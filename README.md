# chihuahua-bot

Slack bot

## Libraries

[Slacker](https://github.com/shomali11/slacker) - Slack Bot Framework
[ansi]("https://github.com/mgutz/ansi") - Pretty terminal colors

## Development

- install `go 1.13`
- install `docker-cli`
- clone this repo
- in the repo, run
  - windows `./start.cmd`
  - linux `./start.sh`
